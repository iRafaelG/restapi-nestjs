# Backend with NestJS

This project was generated with NestJS. Please, before running make sure you have ran your MongoDB services.

A simple  Rest API to CRUD Products.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
